Create some RMD and render it

The idea here is to have a web interface to create an rmd file (or string at least), then send that to php land, process the file or string with R, render a bunch of other files, and serve those to the client

To get started, type some RMarkdown in the text box and submit. Default is fine

This system requires JavaScript.
